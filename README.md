## 📊 Análise de Gastos Públicos — Arraial do Cabo

Este projeto é um dashboard interativo desenvolvido com Streamlit para análise dos gastos públicos da Prefeitura de Arraial do Cabo, utilizando dados reais do portal da transparência.

A aplicação permite visualizar a evolução dos gastos ao longo do tempo, identificar os órgãos e fornecedores com maior volume de despesas e explorar os dados de forma interativa.

O objetivo é transformar dados públicos em informações claras e acessíveis, facilitando a análise e a transparência na utilização dos recursos públicos.
